# Asteroid Blaster

A fast-paced arcade-style shooter built using the **Pygame** library. Navigate your ship, avoid collisions, and blast incoming asteroids to achieve the highest score!

## Features
- Classic top-down space shooter mechanics.
- Increasing difficulty as you destroy more asteroids.
- Smooth controls and retro-style graphics.

## Prerequisites
Before running the game, ensure you have Python installed on your system. You will also need the `pygame` library.

```bash
pip install pygame
```

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/luis-tirado/asteroid_blaster.git
   ```
2. Navigate to the project directory:
   ```bash
   cd asteroid_blaster
   ```

## How to Play
Run the main script to start the game:

```bash
python main.py
```

### Controls
- **Arrow Keys / WASD**: Move your ship.
- **Spacebar**: Fire laser.
- **Esc**: Quit game.
